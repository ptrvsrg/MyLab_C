Вход:
строка 1:
	целое N от 0 до 2000 -- число точек на плоскости
от строки 2 до N+1:
	2 целых числа от INT16_MIN до INT16_MAX -- координаты точки на плоскости

Выход:
	если N не от 0 до 2000, то "bad number of points"

    если координата точки не от INT16_MIN до INT16_MAX, то "bad coordinates"

	если строк меньше N+1, то "bad number of lines"

	иначе координаты точек наименьшего выпуклого многоугольника, для которого каждая точка из заданного 
    множества точек либо лежит на границе этого многоугольника, либо находится в его внутренней области

Исполняемый файл + данные не более 4Mb

Пример 1
Вход:
4
0 0
2 1
4 0
0 4
Выход:
0 0
4 0
0 4

Пример 2
Вход:
3
200000 1
0 300000
400000 500000
Выход:  
bad coordinates

Пример 3:
Вход:
5
1 1
4 5
Выход:
bad number of lines
